1. Debugging Asistido

Prompt:
"Tengo este error en [lenguaje/framework]: [código o mensaje de error]. Actúa como un debugger senior. Analiza: 1) Causa raíz del error, 2) Por qué ocurre (explicación técnica clara), 3) Solución paso a paso, 4) Cómo prevenirlo en el futuro. Si hay múltiples causas posibles, priorízalas por probabilidad."

2. Code Review Automatizado

Prompt:
"Eres un senior developer haciendo code review. Revisa este código: [pega tu código]. Evalúa: 1) Posibles bugs, 2) Problemas de rendimiento, 3) Violaciones de principios SOLID/clean code, 4) Seguridad, 5) Tests faltantes. Dame sugerencias concretas con ejemplos de código mejorado. Sé exigente."

3. Generación de Tests Unitarios

Prompt:
"Genera tests unitarios completos para esta función en [lenguaje]: [código]. Cubre: casos normales, casos borde, errores esperados, y mocks necesarios. Usa [framework de tests]. Los tests deben seguir el patrón AAA (Arrange, Act, Assert). Dame también la cobertura estimada."
PUBLICIDAD · GOOGLE ADSENSE

4. Refactoring de Código Legacy

Prompt:
"Refactoriza este código legacy para mejorar mantenibilidad sin cambiar su funcionalidad: [código]. Aplica: 1) Extracción de funciones, 2) Nombres descriptivos, 3) Eliminación de código muerto, 4) Patrones de diseño si aplican. Explícame cada cambio y por qué mejora el código."

5. Diseño de Arquitectura

Prompt:
"Eres un arquitecto de software. Necesito diseñar un sistema para [requisitos]. Tecnologías disponibles: [stack]. Dame: 1) Diagrama de arquitectura (textual), 2) Componentes y sus responsabilidades, 3) Flujo de datos, 4) Decisiones técnicas clave con justificación, 5) Posibles cuellos de botella y cómo mitigarlos."

6. Explicación de Código Complejo

Prompt:
"Explícame este código como si fuera un developer junior: [código]. Dame: 1) Qué hace a alto nivel, 2) Cómo funciona cada parte, 3) Por qué está estructurado así, 4) Posibles mejoras. Si tiene patrones de diseño, nómbralos y explica por qué se usaron."
PUBLICIDAD · GOOGLE ADSENSE

7. Optimización de Rendimiento

Prompt:
"Analiza el rendimiento de este código: [código]. Identifica: 1) Cuellos de botella (big-O), 2) Operaciones innecesarias, 3) Problemas de memoria, 4) Consultas N+1 o similares. Dame una versión optimizada y explica cuánto mejora cada cambio. Prioriza las optimizaciones de mayor impacto."

8. Documentación Técnica

Prompt:
"Genera documentación técnica completa para este código: [código]. Incluye: 1) Descripción del módulo/función, 2) Parámetros y tipos, 3) Valor de retorno, 4) Excepciones, 5) Ejemplos de uso, 6) Notas de implementación. Formato JSDoc/TSDoc/docstring según el lenguaje."

9. Migración de Código

Prompt:
"Migra este código de [lenguaje/framework A] a [lenguaje/framework B]: [código]. Mantén la misma funcionalidad pero siguiendo las convenciones y mejores prácticas del lenguaje destino. Dame también un plan de migración si hay dependencias que considerar."
PUBLICIDAD · GOOGLE ADSENSE

10. Generación de Scripts y Utilidades

Prompt:
"Crea un script en [lenguaje] que [tarea específica]. Requisitos: [lista]. El script debe: 1) Manejar errores correctamente, 2) Tener logs informativos, 3) Aceptar argumentos CLI, 4) Ser reutilizable. Incluye instrucciones de uso y ejemplo."

11. Consultas SQL Optimizadas

Prompt:
"Eres un DBA experto. Optimiza esta consulta SQL: [código SQL]. Analiza: 1) Índices necesarios, 2) Plan de ejecución estimado, 3) Subconsultas que se pueden evitar, 4) JOINs ineficientes. Dame la versión optimizada y explica por qué es mejor. Base de datos: [PostgreSQL/MySQL/SQL Server]."

12. Resolución de Problemas Técnicos

Prompt:
"Estoy enfrentando este problema técnico en producción: [descripción]. Stack: [tecnologías]. Dame un plan de diagnóstico y resolución: 1) Hipótesis más probable, 2) Pasos para confirmar, 3) Solución temporal (workaround), 4) Solución definitiva, 5) Medidas para evitar recurrencia. Prioriza rapidez de implementación."
