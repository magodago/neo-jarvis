El prompt maestro para tu bandeja de entrada

Copia este prompt en ChatGPT, Claude o Gemini y pega el contenido de los emails que quieras procesar. El prompt hace todo el trabajo:

Prompt principal
Eres mi asistente de correo electronico. Estos son los emails de mi bandeja de entrada: [PEGA EMAILS]. Clasificalos en: 1) URGENTE (requiere accion hoy), 2) PENDIENTE (puede esperar 48h), 3) INFORMATIVO (solo leer), 4) BASURA (spam/newsletter). Para los urgentes, redactame una respuesta lista para enviar. Para los informativos, haz un resumen de una linea.

Este simple prompt transforma 30 minutos de gestion de correo en 2 minutos. Lo usas, pegas los emails, y la IA te devuelve todo clasificado con respuestas listas.

Prompt para respuestas profesionales

Cuando necesites responder un email importante pero no quieras pasar 15 minutos redactando:

Prompt de respuestas
Eres un experto en comunicacion profesional. Necesito responder a este email: [PEGA EMAIL]. Dame 3 versiones de respuesta: 1) Directa y ejecutiva (2 parrafos), 2) Cercana y detallada, 3) Formal con estructura. Mi relacion con el remitente es [cliente/jefe/colega]. Incluye asunto sugerido para cada version.

Prompt para seguimiento semanal

Al final de cada semana, usa este prompt para asegurarte de que no se te escapa nada importante:

Prompt de revision
Eres un gestor de tareas. Revisa estos emails de la semana: [PEGA EMAILS]. Identifica: 1) Promesas hechas que debo cumplir la proxima semana, 2) Personas que estan esperando respuesta mia, 3) Fechas limite que se acercan, 4) Tareas que puedo dejar para la siguiente semana sin riesgo. Genera una lista de pendientes priorizada.
