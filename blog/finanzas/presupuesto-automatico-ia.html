La mayoria de las personas saben que deberian hacer un presupuesto, pero pocos lo hacen porque parece una tarea tediosa. La buena noticia es que con la IA, crear y mantener un presupuesto mensual lleva menos de 5 minutos.

En este tutorial te enseno 3 prompts que transforman tus datos financieros en un presupuesto claro y accionable.

Prompt 1: Analisis de gastos automatico

Pega tus gastos del mes y la IA los clasifica por categoria, identifica patrones y te muestra donde esta el mayor potencial de ahorro.

Prompt
Eres un analista financiero. Clasifica estos gastos del mes en categorias (vivienda, alimentacion, transporte, ocio, suscripciones, etc.): [PEGA GASTOS]. Dame: 1) Porcentaje de ingresos por categoria, 2) Los 3 gastos mas altos, 3) Oportunidades de ahorro identifica-das, 4) Comparativa con el mes anterior si tienes datos.

Prompt 2: Creacion del presupuesto

Con el analisis hecho, este prompt crea un presupuesto personalizado aplicando la regla 50/30/20.

Prompt
Eres un planificador financiero. Con estos datos: ingresos mensuales de [cantidad] y gastos clasificados: [PEGA ANALISIS]. Creame un presupuesto usando la regla 50/30/20 (50% necesidades, 30% deseos, 20% ahorro). Para cada categoria, dame el limite mensual recomendado y alertas si estoy gastando de mas.

Prompt 3: Seguimiento semanal

Cada semana, pasa tus gastos por este prompt para ver si vas encaminado o necesitas ajustar.

Prompt
Eres un controlador de presupuesto. Estos son mis gastos de esta semana: [PEGA GASTOS]. Mi presupuesto mensual es: [PEGA PRESUPUESTO]. Evaluame: 1) Voy por buen camino? (verde/amarillo/rojo), 2) Cuanto me queda para el resto del mes en cada categoria, 3) Ajustes recomendados para la proxima semana, 4) Si sigo este ritmo, cuanto ahorrare este mes.
