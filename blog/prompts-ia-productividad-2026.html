1. El Planificador Diario Perfecto

Empieza cada dia con un plan claro. Este prompt organiza tus tareas usando la matriz Eisenhower en segundos. Dile adios a las mañanas de caos y prioriza lo que realmente importa.

Prompt
Eres un experto en productividad. Basandote en mi lista de tareas, planifica mi dia de forma optima. Prioriza usando matriz Eisenhower, asigna tiempos realistas y dame 3 bloques de enfoque de 90 minutos. TAREAS: [lista] DISPONIBILIDAD: [horas]

2. El Email Profesional en 1 Minuto

Escribe emails que consiguen respuesta sin perder 20 minutos redactando. Este prompt te da 3 versiones para elegir la que mejor se adapte a cada situacion.

Prompt
Eres mi asistente de comunicacion. Necesito un email sobre [tema]. Tono: [profesional/cercano/formal]. Destinatario: [cliente/jefe]. Accion requerida: [que necesito]. Dame 3 versiones: directa, detallada y persuasiva.

3. Resumen Ejecutivo de Documentos

Ahorra horas de lectura extrayendo lo esencial de cualquier texto. Este prompt transforma articulos, informes y documentos en resumenes accionables en segundos.

Prompt
Eres un analista de informacion. Extrae de este texto: 1) Resumen ejecutivo (3 lineas), 2) 5 puntos clave, 3) Datos importantes, 4) Acciones recomendadas. Prioriza precision sobre extension.

4. El Solucionador de Problemas

Cuando estas atascado en una decision, este prompt te da claridad aplicando el metodo de los 5 porque y generando un plan de accion concreto.

Prompt
Eres un consultor estrategico. Mi problema: [describe]. Aplica el metodo de los 5 porque para encontrar la causa raiz, genera 3 opciones viables, evalua pros/contra de cada una y recomienda UNA con plan de accion concreto.

5. Reuniones Productivas

Transforma reuniones que no llevan a ningun lado en sesiones donde se toman decisiones. Este prompt estructura el antes, durante y despues de cada reunion.

Prompt
Eres un facilitador de reuniones. Para mi reunion sobre [tema] con [personas]: 1) Prepara agenda optima para 30min, 2) Define objetivos medibles, 3) Asigna tiempos por punto, 4) Crea template de minutos con acuerdos y responsables, 5) Sugiere follow-up en 48h.

6. Gestion de Proyectos

Organiza proyectos complejos sin perder detalle. Este prompt desglosa cualquier proyecto en tareas manejables con plazos realistas.

Prompt
Eres un project manager senior. Desglosa este proyecto en fases: [nombre]. Para cada fase: 1) Tareas concretas, 2) Duracion estimada, 3) Dependencias, 4) Recursos necesarios, 5) Criterios de exito. Identifica tambien los 3 riesgos principales y como mitigarlos.

7. Brainstorming Estructurado

Genera ideas de calidad en minutos con un proceso que evita el pensamiento grupal y las ideas genericas.

Prompt
Eres un facilitador de innovacion. Necesito ideas para [desafio]. Aplica: 1) Brainstorming libre (10 ideas en 2min), 2) SCAMPER para expandir (Sustituir, Combinar, Adaptar, Modificar, Poner en otro uso, Eliminar, Revertir), 3) Selecciona las 3 ideas mas prometedoras con criterios de viabilidad e impacto.

8. Analisis de Decisiones

Cuando tienes que elegir entre opciones y no sabes cual es la mejor, este prompt aplica un analisis multicriterio objetivo.

Prompt
Eres un analista de decisiones. Ayudame a decidir entre estas opciones: [lista]. Para cada una evalua: 1) Coste, 2) Tiempo, 3) Impacto, 4) Riesgo, 5) Dificultad. Usa una matriz de decision ponderada y recomienda la mejor opcion con justificacion.

9. Delegacion Efectiva

Delegar no es soltar tareas, es multiplicar tu tiempo. Este prompt te ayuda a delegar de forma clara y efectiva.

Prompt
Eres un experto en liderazgo y delegacion. Voy a delegar [tarea] a [persona]. Generame: 1) Briefing claro con contexto y objetivo, 2) Instrucciones paso a paso, 3) Entregables esperados con criterios de calidad, 4) Timeline con checkpoints, 5) Recursos y autonomia que debe tener.

10. Revision Semanal

Termina cada semana con claridad y empieza la siguiente con ventaja. Este prompt estructura tu revision semanal en 15 minutos.

Prompt
Eres un coach de productividad. Guiame en mi revision semanal: 1) Que logre esta semana? (3 exitos), 2) Que aprendi? (2 lecciones), 3) Que puedo mejorar? (1 cosa), 4) Mis 3 prioridades para la proxima semana, 5) Que necesito para avanzar en cada prioridad?
Consejo

Guarda estos prompts en un documento y adaptalos a tu contexto. Cuanto mas especifico seas al completar los [corchetes], mejores resultados obtendras.
